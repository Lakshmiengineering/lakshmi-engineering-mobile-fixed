# Lakshmi Engineering Website

## Folder Structure
```
lakshmi-engineering/
├── index.html          ← Main website file
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Interactivity
├── assets/
│   └── logo.png        ← ← ← PUT YOUR LOGO HERE
└── README.md
```

## How to Add Your Logo

1. Copy your logo PNG file into the `assets/` folder
2. Rename it to `logo.png`
3. Open `index.html` in a browser — your logo will appear automatically in the navbar and footer

> The logo display size in the navbar is **50px tall** (auto width).  
> For best results use a PNG with transparent background.

## How to Open

Simply open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

No build tools or server required — it works as a standalone static website.

## To Deploy

Upload all files as-is to:
- **Hostinger / cPanel** → Upload to `public_html/`
- **Netlify** → Drag & drop the entire folder
- **GitHub Pages** → Push to a repository
